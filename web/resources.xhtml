Where can I get the original association files used for these analyses?

The files for most of the species were obtained from: EBI

Ecoli data (ecocyc) was obtained from: Gene Ontology

Fly data was obtained from: FlyBase (older editions) and EBI (newer editions)

Yeast data was retrieved from: EBI (newer editions) and SGD (older editions)

What files were used to map the identifiers?

Part of the yeast mapping was done with information from the Uniprot website.

For selected organisms, the Uniprot Mapping file was used

However, as some of these files do not have the latest (current) identifier used in many cases, a direct query to the Uniprot website was performed to keep them up-to-date.

Where can I get the mapping from Medline to Pubmed Ids?

Mapping the MEDLINE identifiers with current PMID identifiers was done using the MuID-Pmid(1-6).zip files from NCBI
